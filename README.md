# STRAF to Tercen operator

##### Description

`straf2tercen` operator allows one to import STRAF input data into Tercen.

##### Usage

Input projection|.
---|---
`col`        | character, document ID

Output relations|.
---|---
`allele`        | numeric, allele name
`frequency`        | numeric, allele frequency, per individual (0, 0.5, or 1)
`sample`        | character, individual/sample name
`population`        | character, population name
